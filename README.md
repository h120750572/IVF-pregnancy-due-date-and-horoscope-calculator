# 試管嬰兒懷孕計算機 🤱💕

一個專為試管嬰兒療程設計的智能懷孕計算工具，提供預產期計算、星座預測、產檢衛教和胚胎植入時間規劃功能。

## 🌟 功能特色

### 1. 最後月經日期計算 📅
- 輸入最後月經第一天日期
- 計算當前懷孕週數和天數
- 預測預產期和寶寶星座
- 估算胎兒體重

### 2. 胚胎植入日期計算 🧬
- 支援不同胚胎發育天數（第2天、第3天、第5天囊胚）
- 精確計算試管嬰兒懷孕週數
- 自動換算等效最後月經日期
- 專業的醫學計算邏輯

### 3. 星座倒推功能 ⭐ **NEW!**
- 選擇想要的寶寶星座
- 智能計算建議的胚胎植入時間範圍
- 支援不同胚胎類型（新鮮/冷凍、第3天/第5天）
- 幫助媽媽規劃理想的懷孕時程

### 4. 輔助產檢衛教 📚
- 根據懷孕週數提供專業產檢指導
- 涵蓋第一、二、三孕期重要檢查項目
- 提供生活保健建議和注意事項
- 基於正式產檢衛教手冊內容

## 🎯 目標用戶

- 正在進行試管嬰兒療程的夫妻
- 計劃懷孕的準父母
- 想要「訂製」寶寶星座的家庭
- 需要專業產檢指導的孕婦
- 生殖醫學中心和婦產科診所

## 🛠️ 技術架構

### 前端技術
- **React 18** - 現代化前端框架
- **Vite** - 快速建構工具
- **Tailwind CSS** - 實用優先的CSS框架
- **shadcn/ui** - 高品質UI組件庫
- **Lucide React** - 美觀的圖標庫

### 核心功能模組
- `calculations.js` - 懷孕週數和預產期計算
- `zodiacReverseCalculations.js` - 星座倒推計算邏輯
- `educationData.js` - 產檢衛教資訊數據
- `App.jsx` - 主要應用程式組件

### 計算邏輯
- **LMP計算**：基於最後月經日期的標準懷孕計算
- **胚胎植入計算**：考慮胚胎發育天數的專業計算
- **星座倒推**：從目標星座反推最佳植入時間
- **胎兒體重估算**：基於懷孕週數的標準體重曲線

## 📱 使用說明

### 基本操作
1. 選擇計算方式（最後月經日期、胚胎植入日期、星座倒推）
2. 輸入相關日期或選擇目標星座
3. 點擊「開始計算」獲得結果
4. 查看詳細的計算結果和衛教資訊

### 星座倒推功能
1. 點擊「星座倒推」標籤頁
2. 選擇想要的寶寶星座
3. 選擇目標年份（2025-2027年）
4. 選擇胚胎類型（新鮮/冷凍、第3天/第5天）
5. 獲得建議的植入時間範圍

## 🏥 醫學準確性

本計算機基於以下醫學標準：
- **WHO懷孕週數計算標準**
- **ACOG（美國婦產科學會）指引**
- **台灣婦產科醫學會建議**
- **正式產檢衛教手冊內容**

### 計算公式
- **標準懷孕期**：280天（40週）
- **胚胎植入調整**：
  - 第2天胚胎：LMP = 植入日期 - 16天
  - 第3天胚胎：LMP = 植入日期 - 17天  
  - 第5天囊胚：LMP = 植入日期 - 19天
